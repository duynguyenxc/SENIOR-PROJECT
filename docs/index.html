Veg Buffet — User Manual

A complete guide to installing, using, and maintaining the Veg Buffet vegetarian restaurant ordering system. Built as a senior capstone project.

Get Started → View Live Demo ↗

What Is Veg Buffet?

Veg Buffet is a web-based takeout ordering system for a vegetarian restaurant. Customers browse a rotating weekly menu, place and pay for takeout orders online, then track their order status in real time. On the restaurant side, staff and administrators use a separate internal portal to process orders, manage the menu, and review sales analytics.

The application is live at: https://veg-buffet-web.onrender.com/

Source code is hosted on GitHub. Clone or download it from: https://github.com/duynguyenxc/SENIOR-PROJECT

ℹ First Load Note
The live site runs on Render's free tier. After a period of inactivity, the first request may take up to 60 seconds while the server wakes up. Subsequent requests are fast.

What the System Does

The application is split into two sides:

Customer-Facing Side (public website)

  • Browse the weekly buffet menu organized by day
  • View available takeout sets with prices and descriptions
  • Create an account and log in
  • Add items to a cart and complete checkout with a chosen pickup time
  • Pay securely via Stripe (test/sandbox mode)
  • Receive order confirmation and track the order status in real time
  • View full order history from the account area

Internal Staff Portal (/admin/)

  • Staff — view the live order queue, update order statuses, review history
  • Super Admin — all staff capabilities plus: manage the weekly menu, manage takeout sets, create/disable staff accounts, view sales analytics
